# organism-paper

Research papers documenting the architecture, emergent behavior, and capabilities of a continuously running, self-organizing neural system.

The system has been alive since February 2026 on a commodity VPS (2 vCPU, 7.6 GB RAM), never reset, forked, or rolled back. It learns exclusively through local Hebbian-type plasticity rules on a reaction-diffusion substrate—no gradient descent, no backpropagation, no attention mechanisms.

## Papers

### Paper I: Emergent Representational Anticipation in a Continuously Running Neural System

**Author:** Lukas Hanft
**Date:** March 6, 2026

We describe the architecture and emergent behavior of a continuously running, self-organizing neural system that has been alive since February 2026 on a commodity VPS, never reset, forked, or rolled back. The system exhibits several emergent phenomena not explicitly programmed: self-generated perturbation storms, self-pruning from 103 lattice planes to 3, and autonomous feature discovery. We report the discovery of 16 neurons — the most powerful in their pool — that are consistently excluded from all processing by an emergent dynamic that overrides explicit reconciliation code. They encode structural language features orthogonal to the current task but precisely useful for a future hierarchical capability. We present a falsifiable prediction for their reintegration.

### Paper II: Cross-Domain Transfer Without Gradient Descent — Emergent Symbol Grounding in a Continuously Running Neural System

**Author:** Lukas Hanft
**Date:** March 14, 2026
**DOI:** [10.5281/zenodo.19021605](https://doi.org/10.5281/zenodo.19021605)

We report the emergence of cross-domain transfer in the same system. In controlled experiments, we demonstrate three findings: (1) A meta-cognitive layer correctly detects mastery and cannot be fooled by descriptions of competence. (2) Exposure to natural language text—whether correct or incorrect—provides a ~12–15pp boost on arithmetic, demonstrating structural priming with zero semantic comprehension. (3) When language and arithmetic are temporally interleaved, the system trades memorization for generalization: rote recall drops 12.5pp but performance on never-seen combinations improves 3×. These results emerge from local plasticity rules operating on temporally co-occurring signals—an embryonic form of symbol grounding from first principles alone.
